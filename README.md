# ワイヤーフレーム実践ガイド

図解、スライド、Web、アプリ、広告バナー、教材制作に使えるワイヤーフレーム実践ガイドです。

## 見る方法

`index.html` をブラウザで開いてください。外部ライブラリやビルドは不要です。

## 内容

- ワイヤーフレームの基本
- 5段階の作成プロセス
- 小学校教員、行政職、デザイナー、Webエンジニア、個人事業主、教材製作者向けの解説
- レイアウトパターン一覧
- よくある失敗例と成功例
- 下準備と注意点
- 人間と生成AIの分担
- レビュー用チェックリスト
- AI依頼用プロンプト
